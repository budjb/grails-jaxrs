Package: org.grails.plugins.jaxrs.provider

[Groovy] Class JSONWriter

        • org.grails.plugins.jaxrs.provider.JSONWriter

        • Provider for Grails' grails.converters.JSON converter class. Message body writer that supports grails.converters.JSON response entities. For example, to create a JSON representation from a list of Grails domain objects: 

           @Path('/notes')
 @Produces('text/x-json') // or 'application/json'
 class NotesResource {*
      @GET
      XML findNotes() {*          Note.findAll() as JSON
}*
}*
          Alternatively, one could write 
           @Path('/notes')
 @Produces('text/x-json') // or 'application/json'
 class NotesResource {*
      @GET
      Response findNotes() {*          Response.ok(Note.findAll() as JSON).build()
}*
}*

          Martin Krasser
