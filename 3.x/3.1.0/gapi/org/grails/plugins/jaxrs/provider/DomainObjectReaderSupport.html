[Groovy] Class DomainObjectReaderSupport

    • org.grails.plugins.jaxrs.provider.DomainObjectReaderSupport
    • All Implemented Interfaces and Traits:
      javax.ws.rs.ext.MessageBodyReader, grails.core.support.GrailsApplicationAware

      abstract class DomainObjectReaderSupport
extends java.lang.Object

      A message body reader that converts an XML or JSON entity streams to a domain object. The request content type must be specified by the Content-Type HTTP header. Supported content types by this provider are text/xml, application/xml, text/x-json or application/json. Assuming Note is a Grails domain class. This provider supports usage of resource methods such as: 

       @Path('/notes')
 class NotesResource {*
      @POST
      @Consumes(['application/xml','application/json'])
      Response addNote(Note note) {*          note.save()
          // ...
}*
}*
