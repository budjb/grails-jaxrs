Package: org.grails.plugins.jaxrs.provider

[Groovy] Class DomainObjectWriterSupport

    • org.grails.plugins.jaxrs.provider.DomainObjectWriterSupport
    • All Implemented Interfaces and Traits:
      javax.ws.rs.ext.MessageBodyWriter, grails.core.support.GrailsApplicationAware

      abstract class DomainObjectWriterSupport
extends java.lang.Object

      A message body writer that converts Grails domain objects to XML or JSON entity streams. The expected response content type is specified by the Accept request header. Supported content types by this provider are text/xml, application/xml, text/x-json or application/json. Assuming Note is a Grails domain class. This provider supports usage of resource methods such as: 

       @Path('/notes')
 class NotesResource {

      @GET
      @Produces(['application/xml','application/json'])
      Note getNote(...) {
          Note note = ... // e.g. load a note object from database.
          return note
      }

      @GET
      @Produces(['application/xml','application/json'])
      Collection<Note> getNotes(...) {
          def notes = ... // e.g. load list of note objects from database.
          return notes
      }

      @GET
      @Produces(['application/xml','application/json'])
      def getNotes(...) {
          def notes = ... // e.g. load list of note objects from database.
          return notes
      }
 }
