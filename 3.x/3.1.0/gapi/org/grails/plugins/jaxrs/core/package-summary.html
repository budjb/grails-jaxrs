Package org.grails.plugins.jaxrs.core

  • Interface Summary 
    Interface Description
    ResourceRegistrar An interface defining a class that is responsible for providing the plugin with a list of resource classes to add the JaxrsContext during application initialization.
  • Class Summary 
    Class Description
    JaxrsApplicationConfig Defines the components (resource and provider classes) of a JAX-RS applications.
    JaxrsContext The JAX-RS context used by applications to interact with a JAX-RS implementation.
    JaxrsFilter A servlet filter that stores the request URI as request attribute.
    JaxrsListener Initializes a Jersey servlet for request processing.
    JaxrsRequestWrapper A request wrapper that decorates javax.servlet.http.HttpServletRequest#getRequestURI().
    JaxrsServletConfig A servlet configuration that allows for modification of the class list.
    JaxrsUtil Provides utility methods to the JAX-RS plugin.
    ScanningResourceRegistrar An implementation of ResourceRegistrar that provides classpath scanning abilities to the registrar.
